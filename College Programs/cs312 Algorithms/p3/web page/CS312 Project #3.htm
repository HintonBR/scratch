CS 312 -- Project #3 -- Dynamic Programming

General Information for Projects 3, 4, and 5:

The Traveling Salesperson Problem (TSP).

A salesperson must tour n cities exactly once and return to the city from which she started. We treat the problem as a tour in a weighted directed complete graph on n vertices.
The problem is to find a tour whose distance is smallest.

Data file format: Projects 3, 4, and 5.

The data file DIFFERS from the file format of the first two projects. In the first two projects, each data line contained two pieces of data.
In projects 3, 4, and 5, each data line after the first line will contain n pieces of data, where n is the number of datalines in the file.
The jth element on the ith line is the distance from city i to city j. Distances need NOT be symmetric!
Each data element will satisfy:
0 <= element <= 1,000
